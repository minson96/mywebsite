src와 width는 attribute(속성을 의미한다.)
<ul> unorderd list or <ol> ordered list 로 ordered list는 자동으로 순서를 매겨준다.
		<li></li> => list 목록을 의미하며 <ul> 이나 <ol> 안에 존재해야 한다.
</ul> or </ol>
<title></title> => 페이지 제목을 의미한다.
<head></head> => 본문 설명을 의미한다.
<body></body> => 본문
<html></html> => 최상단에 위치하는 태그이다.
<!doctype html> => 관용적으로 사용되며 html로 작성되었다는 것을 말하기 위해 첫 줄에 작성한다.
<a href="사이트 경로" title="경로 설명" target="_blank"></a> => anchor 닻이라는 의미로 
hyper reference로 이동시킬 수 있다. 이때 target에서 blank는 새 탭에서 실행하는 것을 의미한다.

최종 정리 형태

<!doctype html>
<html>
    <head>
        <title>구글</title>
        <meta charset="utf-8">
    </head>
    <body>
        <h1><a href="index.html">구글</a></h1>
        <ol>
            <li><a href="index.html">구글</a></li>
        </ol>
        <h2><a href="https://www.google.com/"
            target="_blank" title="구글 이동">구글 이동</a></h2>
        <p>구글이다.</p>
        <img src="1.jpg" width="100%">
        <p><strong><u>구글</u>사이트이다.</strong><br>
        1. 기름기를 없앤 갈빗살에 칼집을 <br>
        </p>
    </body>
</html>
